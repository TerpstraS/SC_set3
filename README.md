# SC_set3
Scientific Computing Set 3

## Files
- `main.py` - entry point for program to execute all code, except for the bonus question
- `leapfrog.py` - entry point for the bonus question

## Usage
To install all necessary packages, run:
```
  pip3 install requirements.txt
```

Run all models:
```
python3 main.py
```
or
```
python3 leapfrog.py
```
